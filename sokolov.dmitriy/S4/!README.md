# Available commands

1. `HELP` <br>
   Prints this list
2. `INSERT <ENGLISH WORD> <TRANSLATION>` <br>
   Adds a translation to the dictionary
3. `RMTR <ENGLISH WORD> <TRANSLATION>` <br>
   Removes a translation from the dictionary
4. `RMEW <ENGLISH WORD>` <br>
   Removes an English word and all its translations from the dictionary
5. `SEARCH <ENGLISH WORD> <TRANSLATION>` <br>
   Searches if the translation of a word exists in the dictionary
6. `GET <ENGLISH WORD>` <br>
   Gets the translations of a word from the dictionary
7. `NON` <br>
   Gets the number of nodes in the tree
8. `OUT` <br>
   Prints the whole tree
9. `LOAD <FILENAME>` <br>
   Loads translations from a file
10. `QUIT` or EOF <br>
   Quits the program
